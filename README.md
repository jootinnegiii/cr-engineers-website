# C.R. Engineers Technologies

Static company profile website — layout inspired by [HopeRise Charity Landing Page](https://dribbble.com/shots/26290772-HopeRise-Charity-Landing-Page-Design).

## Run locally

```bash
python -m http.server 8080
```

Open [http://localhost:8080](http://localhost:8080).

## Files

- `index.html` — all company profile content
- `styles/main.css` — design & responsive layout
- `scripts/main.js` — mobile navigation
